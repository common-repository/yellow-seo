Adds  META tags for the titles, descriptions, keywords and tags for facebook opengraph and twitter cards for all pages and posts and any CPT. You can also set default values for the frontpage or home page. Includes Google webmaster tools (site verification) and Google analytic.

== Description ==

* Nonce Security
* Generates META tags
* Generates facebook opengraph tags
* Generates twitter card tags
* Sets the featured image as a tag for facebook and twitter 
* Google Analytic
* Google Webmaster Tools

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/yellow-seo` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> Yellow SEO screen to configure the META info for the homepage, Google webmaster tools, and Google analytic.
